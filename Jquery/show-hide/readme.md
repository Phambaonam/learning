Node.removeChild



Node.removeChild() 方法从DOM中删除一个子节点。返回删除的节点。

语法 
let oldChild = node.removeChild(child);

//OR

element.removeChild(child);


child 是要移除的那个子节点.
node 是child的父节点.
oldChild保存对删除的子节点的引用. oldChild === child.
